Overview

Implement a TODO list API using your favorite database.
You can choose only one type of database.
You can use any database schema, but your must initialize it in your code.

Notes

  • <%- NOTE.READY %>
  • <%- NOTE.MULTI_INSTANCE %>
  • For 400 and 404 responses you can return any response body (it can be empty or contain any JSON).
  • id must be auto generated by your app (auto increment integer, starts from 1).
